Σχετικά

Αυτή είναι μια στατική σελίδα σε single HTML. Ιδανικό για GitHub Pages. Το chat λειτουργεί με localStorage.

Οδηγίες ανέβασμα στο GitHub Pages:

  1. Δημιούργησε repo με όνομα username.github.io ή όποιο repo
  2. Πρόσθεσε αυτό το αρχείο ως index.html στο root
  3. Στα Settings του repo ενεργοποίησε GitHub Pages (branch main / root). Η σελίδα θα εμφανιστεί σε λίγα λεπτά.

Σύντομη περιγραφή

Δύο ομάδες, πλήρης παρουσίαση, και ένα απλό chat που διατηρεί τα μηνύματα τοπικά.

Κλειδί:
  • index.html — root file για GitHub Pages
  • Chat: localStorage (browser)
  • Για κοινό chat: απαιτείται backend

Επιλογές μηνυμάτων

Μπορείς να εξάγεις/εισάγεις το αρχείο JSON με όλα τα μηνύματα — χρήσιμο για backup.

Σημείωση απορρήτου

Τα μηνύματα αποθηκεύονται τοπικά. Αν θέλεις κοινό chat, θα χρειαστείς υπηρεσία τρίτου μέρους ή server.
